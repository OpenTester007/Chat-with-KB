# AI-based Translator (中英日韩四国语言互译与AI智能助手)

基于 AI 的中英日韩四国语言互译与智能助手，提供快速翻译、文本润色、双语词典与 AI 聊天等功能。

> [!NOTE]
> **Microsoft Edge 商店已发布**
> 目前该版本已正式发布至 [Microsoft Edge 扩展商店](https://microsoftedge.microsoft.com/addons/)。

## 🌟 主要功能

- **AI 智能翻译**：支持中、英、日、韩四国语言互译，自动检测源语言，翻译结果准确自然。
- **文本润色（Polish）**：智能识别文本语言，在保留原意的前提下润色文笔，并提供详尽的中文修改说明。
- **AI 词典**：提供核心释义、主要词性、词根词缀深度解析以及中英对照例句。
- **AI 对话聊天**：快捷方便的侧边栏式 AI 聊天对话助手。
- **历史记录**：自动保存最近 10 条翻译与查询历史，方便随时查看。

## ⚙️ 配置与使用

1. 打开扩展面板，进入 **设置**（Settings）页面。
2. 配置您的 API 密钥（默认已配置开箱即用的免费通道）、API 端点及模型名称。
3. 支持流式传输（逐字显示）与非流式传输的灵活切换。
