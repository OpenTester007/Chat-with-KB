AI-based Translator — Privacy Policy

Last updated: June 21, 2026

1. Data Collection

AI-based Translator does not collect any personal user data.

All data (including translation history, API settings, and chat records) is stored exclusively in the user's local browser via the Chrome Storage API and is never uploaded to any third-party server operated by the extension developer.

2. Third-Party Services

This extension uses the NVIDIA Build free API platform by default for translation and chat functionality. User-input text is transmitted via HTTPS to NVIDIA servers for processing. Users may replace this with any compatible API endpoint in the Settings page.

Please review NVIDIA's privacy policy for their data handling practices:

  • NVIDIA Privacy Policy

3. Data Storage

All data is stored locally in the browser:

  • API key (replaceable with your own key)
  • Model name and API endpoint URL
  • Translation history (last 10 records)
  • Chat history (last 5 conversation turns)

Clearing browser data or uninstalling the extension will delete all the above information.

4. Permissions Explained

  • storage — Used to persist user settings and local history.
  • host_permissions — Required to call the NVIDIA translation API and any user-configured custom endpoints.

5. Contact

For privacy-related questions, please contact the developer via the Edge Add-ons store or through the project's GitHub Issues.
