AI-based Translator - Privacy Policy

Last updated: June 23, 2026

1. Data Collection

AI-based Translator does not collect personal user data on servers operated by the extension developer.

API settings, translation history, and chat history are stored locally in the user's browser via the Chrome Storage API.

2. Third-Party Services

This extension sends user-entered text to the API endpoint configured in Settings for translation, dictionary, polishing, or chat responses. The default endpoint is NVIDIA Build's OpenAI-compatible API, but users must provide their own API key or configure another compatible provider.

Remote API endpoints must use HTTPS. Local development endpoints are allowed only for localhost or 127.0.0.1.

Please review the privacy policy of the provider you configure. For NVIDIA Build, see:

  • NVIDIA Privacy Policy

3. Local Data Storage

The following data may be stored locally in the browser:

  • API key, API endpoint, and model name
  • Translation history, limited to the last 10 records
  • Chat history, limited to the last 5 conversation turns

Clearing browser data or uninstalling the extension will delete this local data.

4. Permissions Explained

  • storage: Persists user settings and local history.
  • host permissions: Allows requests to the default API endpoint.
  • optional host permissions: Allows users to grant access to a custom HTTPS API endpoint or local development endpoint when they save settings.

5. Contact

For privacy-related questions, please contact the developer through the project's GitHub Issues.
