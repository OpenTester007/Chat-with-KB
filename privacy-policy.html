AI-based Translator 隐私政策

更新日期：2026年6月21日

1. 数据收集声明

AI-based Translator 不收集任何用户个人数据。

本扩展的所有数据（包括翻译历史、API 设置、对话记录）均仅存储于用户的本地浏览器（通过 Chrome Storage API），不会上传至任何第三方服务器。

2. 第三方服务

本扩展默认使用 NVIDIA Build 免费 API 平台进行翻译和对话。用户输入的文本内容会通过 HTTPS 加密发送至 NVIDIA 服务器进行处理。用户可在「设置」页自行替换为其他兼容的 API 端点。

请查阅 NVIDIA 的隐私政策以了解其数据处理方式：

  • NVIDIA Privacy Policy

3. 数据存储

所有数据存储在本地浏览器中：

  • API 密钥（可替换为用户的个人密钥）
  • 模型名称、API 端点地址
  • 翻译历史（最近 10 条）
  • 对话记录（最近 5 轮）

清除浏览器数据或卸载扩展将删除上述所有信息。

4. 权限说明

  • storage：用于保存用户设置和翻译历史
  • host_permissions：用于调用 NVIDIA 翻译 API（用户可自定义其他端点）

5. 联系我们

如有任何隐私相关问题，请通过 Edge 商店留言或发送邮件至开发者。
